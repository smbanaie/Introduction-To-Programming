# جلسه ۶: نمایش متن – ASCII و یونیکد

## مرور جلسه

حروف، اعداد و نمادها چگونه در کامپیوترها ذخیره می‌شوند؟ امروز سیستم‌های کدگذاری کاراکتر را بررسی خواهیم کرد، از ASCII تا یونیکد، و درک خواهیم کرد که چرا زبان‌های متعدد نیاز به رویکردهای مختلف دارند.

## اهداف یادگیری کلیدی

- توضیح اینکه چگونه کاراکترها به صورت اعداد ذخیره می‌شوند
- تفاوت‌گذاری بین ASCII و یونیکد
- توصیف اینکه چرا یونیکد مورد نیاز است
- ارائه مثال‌هایی از کدگذاری در استفاده روزمره
