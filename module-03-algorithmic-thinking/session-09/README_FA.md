# جلسه ۹: شبه‌کد و ساختارهای کنترل اساسی

## مرور جلسه

امروز از زبان طبیعی به شبه‌کد ساختار یافته حرکت می‌کنیم، یادگیری ساختارهای دنباله، تصمیم (if) و تکرار (حلقه) که پایه همه برنامه‌نویسی را تشکیل می‌دهند.

## اهداف یادگیری کلیدی

- نوشتن الگوریتم‌های ساده در شبه‌کد
- استفاده از دنباله، انتخاب و تکرار
- بیان الگوریتم‌های کلاسیک (حداکثر، جستجو) در شبه‌کد
- آماده‌سازی برای نگاشت شبه‌کد به پایتون
