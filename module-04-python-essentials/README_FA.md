# ماژول ۴: ضروریات پایتون

## نمای کلی

این ماژول مقدمه جامعی بر مبانی برنامه‌نویسی پایتون ارائه می‌دهد. دانشجویان نحو پایه پایتون، ساختارهای داده، جریان کنترل و سازه‌های برنامه‌نویسی ضروری برای نوشتن برنامه‌های پایتون functional را یاد خواهند گرفت.

## اهداف یادگیری

تا پایان این ماژول، دانشجویان قادر خواهند بود:

- محیط توسعه پایتون را راه‌اندازی و تنظیم کنند
- برنامه‌های پایه پایتون را بنویسند، اجرا کنند و اشکال‌زدایی کنند
- انواع داده داخلی پایتون و ساختارها را درک و استفاده کنند
- جریان کنترل را با استفاده از دستورات شرطی و حلقه‌ها پیاده‌سازی کنند
- عملیات ورودی/خروجی را برای تعامل کاربر مدیریت کنند
- مفاهیم برنامه‌نویسی fundamental را در نحو پایتون اعمال کنند
- برنامه‌های پایه پایتون را اشکال‌زدایی و عیب‌یابی کنند

## ساختار ماژول

### جلسه ۱۰: شروع کار با پایتون – محیط و اولین گام‌ها
- نصب و راه‌اندازی محیط پایتون
- استفاده از shell تعاملی پایتون (REPL)
- نوشتن و اجرای اولین اسکریپت‌های پایتون
- نحو و ساختار پایه پایتون

### جلسه ۱۱: متغیرهای پایتون و انواع داده
- اعلام و تخصیص متغیر
- انواع داده داخلی پایتون (اعداد صحیح، اعشاری، رشته‌ها، بولی‌ها)
- تبدیل نوع و casting
- مدیریت حافظه و حوزه متغیر

### جلسه ۱۲: ورودی، خروجی و عبارت‌ها
- ورودی کنسول با استفاده از تابع `input()`
- خروجی قالب‌بندی شده و قالب‌بندی رشته
- عبارت‌های ریاضی و تقدم عملگر
- تبدیل نوع برای ورودی کاربر

### جلسه ۱۳: دستورات شرطی و منطق بولی
- ساختارهای شرطی `if/elif/else`
- عبارت‌های بولی و عملگرهای منطقی
- دستورات شرطی تو در تو
- ارزیابی مقادیر truthy/falsy

### جلسه ۱۴: حلقه‌ها و تکرار
- حلقه‌های `for` برای تکرار دنباله
- حلقه‌های `while` برای تکرار شرطی
- دستورات کنترل حلقه (`break`, `continue`)
- فهم لیست و عبارت‌های generator

### جلسه ۱۵: کار با رشته‌ها
- ایجاد، ایندکس‌گذاری و برش رشته‌ها
- روش‌ها و عملیات رشته
- تکنیک‌های قالب‌بندی رشته
- پردازش و دستکاری متن

### جلسه ۱۶: مجموعه‌های پایتون
- لیست‌ها: دنباله‌های مرتب و قابل تغییر
- تاپل‌ها: دنباله‌های مرتب و غیرقابل تغییر
- دیکشنری‌ها: نگاشت‌های کلید-مقدار
- مجموعه‌ها: مجموعه‌های نامرتب از عناصر منحصر به فرد

## مفاهیم کلیدی

### مبانی نحو پایتون
- **دستورات و عبارت‌ها**: بلوک‌های سازنده برنامه‌های پایتون
- **تورفتگی**: استفاده پایتون از فضای خالی برای ساختار کد
- **کامنت‌ها**: مستندسازی و توضیح کد
- **قراردادهای نام‌گذاری**: دستورالعمل‌های سبک PEP 8

### انواع داده و ساختارها
- **انواع primitive**: `int`, `float`, `str`, `bool`
- **انواع دنباله**: `list`, `tuple`, `range`
- **انواع نگاشت**: `dict`
- **انواع مجموعه**: `set`, `frozenset`

### جریان کنترل
- **اجرای شرطی**: تصمیم‌گیری در برنامه‌ها
- **تکرار**: عملیات repetitive با حلقه‌ها
- **مدیریت استثنا**: مدیریت خطاهای runtime
- **جریان برنامه**: اجرای ترتیبی، شرطی و تکراری

## محیط توسعه پایتون

### ابزارهای مورد نیاز
- **مفسر پایتون**: نسخه ۳.۸ یا بالاتر
- **ویرایشگر کد**: VS Code، PyCharm یا مشابه
- **ترمینال**: برای اجرای اسکریپت‌های پایتون
- **مدیریت بسته پایتون**: pip برای نصب بسته‌ها

### جریان کاری توسعه
1. **نوشتن**: ایجاد اسکریپت‌های پایتون با پسوند `.py`
2. **اجرا**: اجرای اسکریپت‌ها از طریق خط فرمان یا IDE
3. **اشکال‌زدایی**: شناسایی و رفع خطاهای کد
4. **تست**: تأیید رفتار برنامه با ورودی‌های مختلف
5. **refactor**: بهبود ساختار و کارایی کد

## بهترین روش‌های برنامه‌نویسی

- ارائه پیام‌های خطای معنادار
- تست برنامه‌ها با موارد edge
