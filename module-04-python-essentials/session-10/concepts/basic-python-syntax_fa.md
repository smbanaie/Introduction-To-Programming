## اولین برنامه پایتون شما

بیایید با برنامه کلاسیک "سلام، دنیا!" شروع کنیم:

```python
print("سلام، دنیا!")
```

**این چه کاری انجام می‌دهد:** متن "سلام، دنیا!" را روی صفحه نمایش می‌دهد.

خودتان امتحان کنید! مفسر پایتون را باز کنید یا یک فایل `.py` بسازید و آن را اجرا کنید.

---

## قانون خاص پایتون: تورفتگی مهم است

پایتون منحصربه‌فرد است چون از **تورفتگی** (فاصله در ابتدای خطوط) برای سازماندهی کد استفاده می‌کند.

### مثال خوب (تورفتگی صحیح)

```python
if age >= 18:
    print("شما بزرگسال هستید")
    print("می‌توانید رأی دهید")
```

### مثال بد (تورفتگی اشتباه)

```python
if age >= 18:
print("شما بزرگسال هستید")      # ❌ اشتباه! فاصله گم شده
    print("می‌توانید رأی دهید")       # ❌ اشتباه! فاصله ناسازگار
```

### قانون
- برای هر سطح تورفتگی از **۴ فاصله** استفاده کنید (بیشتر ویرایشگرها این کار را خودکار انجام می‌دهند)
- همیشه در فاصله‌گذاری ثابت باشید

## کامنت‌ها: یادداشت‌ها برای انسان‌ها

کامنت‌ها متنی هستند که پایتون نادیده می‌گیرد. از آنها برای توضیح کد خود استفاده کنید.

```python
# این یک کامنت است - پایتون این خط را نادیده می‌گیرد
print("سلام")  # این کامنت در انتهای خط است

# می‌توانید چند کامنت داشته باشید
# مثل این
# برای توضیح چیزی طولانی‌تر
```

**چرا از کامنت استفاده کنیم؟**
- برای توضیح آنچه کد شما انجام می‌دهد
- برای یادآوری به خودتان بعداً
- برای غیرفعال کردن موقت کد بدون حذف آن

## متغیرها: ذخیره داده

متغیرها مثل **جعبه‌های برچسب‌دار** هستند که اطلاعات را در آنها نگه می‌دارید.

### ساختن متغیرها

```python
name = "علی"         # یک جعبه برچسب‌دار "name" حاوی "علی"
age = 25             # یک جعبه برچسب‌دار "age" حاوی 25
height = 1.65        # یک جعبه برچسب‌دار "height" حاوی 1.65
is_student = True    # یک جعبه برچسب‌دار "is_student" حاوی True
```

### قوانین نام‌گذاری متغیرها

**✅ نام‌های خوب:**
```python
user_name = "حسن"
total_count = 100
favorite_color = "آبی"
```

**❌ نام‌های نامعتبر (باعث خطا می‌شوند):**
```python
2nd_place = "نقره"        # نمی‌تواند با عدد شروع شود
my-name = "علی"           # نمی‌توان از خط تیره استفاده کرد
my name = "حسن"           # نمی‌توان از فاصله استفاده کرد
class = "پایتون"          # نمی‌توان از کلمات رزرو شده استفاده کرد
```

### نکات برای نام‌های خوب
- از نام‌های توصیفی استفاده کنید (`total_price` بهتر از `tp` است)
- از حروف کوچک با زیرخط استفاده کنید (`snake_case`)
- ثابت باشید

## گرفتن ورودی از کاربران

تابع `input()` به برنامه شما اجازه می‌دهد از کاربر اطلاعات بگیرد:

```python
name = input("اسمت چیه؟ ")
print(f"سلام، {name}!")
```

**مهم:** `input()` همیشه متن (رشته) به شما می‌دهد. اگر می‌خواهید اعداد:

```python
# تبدیل رشته به عدد صحیح
age = int(input("چند سال داری؟ "))

# تبدیل رشته به عدد اعشاری
height = float(input("قدت به متر: "))
```

## اشتباهات رایج مبتدی‌ها

### اشتباه ۱: گیومه‌های ناسازگار

```python
# ❌ اشتباه
print("سلام')
print('سلام")

# ✅ درست
print("سلام")
print('سلام')
```

### اشتباه ۲: تورفتگی اشتباه

```python
# ❌ اشتباه
if age > 18:
print("بزرگسال")

# ✅ درست
if age > 18:
    print("بزرگسال")
```

### اشتباه ۳: استفاده از = به جای == برای مقایسه

```python
# ❌ اشتباه (= برای انتساب است)
if x = 5:
    print("x برابر 5 است")

# ✅ درست (== برای مقایسه است)
if x == 5:
    print("x برابر 5 است")
```

### اشتباه ۴: فراموش کردن پرانتز

```python
# ❌ اشتباه
print "سلام"

# ✅ درست
print("سلام")
```

### اشتباه ۵: حساسیت به حروف

```python
# ❌ اشتباه
Print("سلام")
PRINT("سلام")

# ✅ درست
print("سلام")
```

پایتون به حروف حساس است! `print` و `Print` متفاوت هستند.

## خودتان امتحان کنید: تمرین‌ها

### تمرین ۱: سلام ساده

یک برنامه بسازید که:
1. نام کاربر را بپرسد
2. سن او را بپرسد
3. یک سلام با هر دو اطلاعات چاپ کند

```python
name = input("اسمت چیه؟ ")
age = input("چند سال داری؟ ")
print(f"سلام {name}! تو {age} سال داری.")
```

### تمرین ۲: ماشین‌حساب ساده

یک برنامه بسازید که:
1. دو عدد را بپرسد
2. آنها را با هم جمع کند
3. نتیجه را نشان دهد

```python
num1 = int(input("عدد اول را وارد کن: "))
num2 = int(input("عدد دوم را وارد کن: "))
result = num1 + num2
print(f"مجموع: {result}")
```

### تمرین ۳: رفع خطاها

این برنامه ۳ خطا دارد. آیا می‌توانید آنها را پیدا و رفع کنید؟

```python
# اصلی (با خطا)
name = input("اسمت چیه؟ )
Age = input("چند سال داری؟ )
print("سلام " + name + تو " + age + " سال داری")
```

## مرجع سریع

| مفهوم | مثال | توضیح |
|-------|------|-------|
| چاپ | `print("سلام")` | نمایش متن روی صفحه |
| متغیر | `x = 5` | ذخیره یک مقدار |
| ورودی | `name = input("نام؟ ")` | گرفتن ورودی کاربر |
| رشته | `"سلام"` یا `'سلام'` | داده متنی |
| عدد صحیح | `42` | عدد کامل |
| عدد اعشاری | `3.14` | عدد اعشاری |
| کامنت | `# این یک یادداشت است` | توسط پایتون نادیده گرفته می‌شود |

---

## نکات کلیدی

1. **پایتون خواناست** - تقریباً مثل انگلیسی به نظر می‌رسد
2. **تورفتگی مهم است** - از ۴ فاصله برای بلوک‌های کد استفاده کنید
3. **متغیرها داده را ذخیره می‌کنند** - مثل جعبه‌های برچسب‌دار
4. **انواع داده مختلف** - متن، اعداد، درست/نادرست
5. **ورودی/خروجی** - از `input()` و `print()` برای ارتباط استفاده کنید
6. **کامنت‌ها کد را توضیح می‌دهند** - با `#` شروع می‌شوند
