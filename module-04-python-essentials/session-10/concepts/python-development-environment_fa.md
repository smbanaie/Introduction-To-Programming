# محیط توسعه پایتون: ابزارهای برنامه‌نویسی

## چه چیزهایی یاد خواهید گرفت
- چه ابزارهایی برای نوشتن کد پایتون نیاز دارید
- چگونه یک ویرایشگر کد انتخاب کنید
- چگونه برنامه‌های پایتون را اجرا کنید

---

## آنچه برای شروع کدنویسی نیاز دارید

برای نوشتن و اجرای برنامه‌های پایتون، به این موارد نیاز دارید:

1. **پایتون** روی کامپیوتر شما (مفسر)
2. یک **ویرایشگر کد** (جایی که کد را می‌نویسید)
3. یک **ترمینال/خط فرمان** (جایی که کد را اجرا می‌کنید)

همین! بیایید به هر کدام نگاه کنیم.

## ۱. نصب پایتون

### بررسی اینکه آیا پایتون قبلاً نصب شده

ترمینال/خط فرمان خود را باز کنید و تایپ کنید:

```bash
python --version
```

یا در برخی سیستم‌ها:

```bash
python3 --version
```

اگر چیزی مثل `Python 3.9.7` را دیدید، آماده‌اید!

### نصب پایتون (اگر ندارید)

#### ویندوز
1. به [python.org/downloads](https://python.org/downloads) بروید
2. روی "Download Python" (دکمه زرد بزرگ) کلیک کنید
3. نصب‌کننده را اجرا کنید
4. **مهم:** گزینه "Add Python to PATH" را در حین نصب علامت بزنید
5. روی "Install Now" کلیک کنید

#### مک
1. ترمینال را باز کنید
2. Homebrew را نصب کنید (اگر ندارید): به [brew.sh](https://brew.sh) بروید
3. تایپ کنید: `brew install python3`

#### لینوکس
معمولاً پایتون از قبل نصب است. اگر نه:

```bash
# Ubuntu/Debian
sudo apt update
sudo apt install python3

# بررسی نصب
python3 --version
```

## ۲. انتخاب یک ویرایشگر کد

ویرایشگر کد مثل یک پردازشگر متن برای برنامه‌نویسی است. گزینه‌های خوبی وجود دارد:

### برای مبتدیان: IDLE (همراه پایتون)

**IDLE** وقتی پایتون را نصب می‌کنید همراه آن می‌آید. ساده است و برای یادگیری عالی.

**چگونه IDLE را باز کنیم:**
- **ویندوز:** "IDLE" را در منوی Start جستجو کنید
- **مک:** در Applications → Python پیدا کنید
- **لینوکس:** `idle` را در ترمینال تایپ کنید

**ویژگی‌ها:**
- رابط ساده
- کد شما را رنگی نشان می‌دهد (برجسته‌سازی نحو)
- راهنمای داخلی برای مبتدیان

### برای کاربران متوسط: VS Code

**VS Code** (Visual Studio Code) رایگان، قدرتمند و محبوب در بین برنامه‌نویسان است.

**چگونه آن را بگیریم:**
1. از [code.visualstudio.com](https://code.visualstudio.com) دانلود کنید
2. آن را نصب کنید
3. افزونه پایتون را نصب کنید (VS Code این را پیشنهاد می‌دهد)

**چرا VS Code؟**
- رایگان و روی همه پلتفرم‌ها کار می‌کند
- با پیشنهادها به شما کمک می‌کند کد بهتری بنویسید
- ترمینال داخلی دارد

### گزینه‌های دیگر

| ویرایشگر | بهترین برای | چگونه بگیریم |
|----------|-------------|---------------|
| **Thonny** | مبتدیان کامل | [thonny.org](https://thonny.org) |
| **PyCharm** | توسعه حرفه‌ای | [jetbrains.com/pycharm](https://jetbrains.com/pycharm) |
| **Sublime Text** | ویرایش سریع، سبک | [sublimetext.com](https://sublimetext.com) |

## ۳. استفاده از ترمینال/خط فرمان

ترمینال (که خط فرمان، کنسول یا shell هم نامیده می‌شود) جایی است که دستورات را برای اجرای برنامه‌های خود تایپ می‌کنید.

### باز کردن ترمینال

- **ویندوز:** `Win + R` را بزنید، `cmd` را تایپ کنید، Enter را بزنید
  - یا از PowerShell استفاده کنید (مدرن‌تر)
- **مک:** `Cmd + Space` را بزنید، `terminal` را تایپ کنید، Enter را بزنید
- **لینوکس:** `Ctrl + Alt + T` را بزنید

### دستورات پایه ترمینال

اینها به شما کمک می‌کنند به فایل‌های پایتون خود برسید:

```bash
# ببینید در چه پوشه‌ای هستید
pwd          # مک/لینوکس
cd           # ویندوز

# فایل‌ها در پوشه فعلی را لیست کنید
ls           # مک/لینوکس
dir          # ویندوز

# به پوشه دیگر بروید
cd Documents
cd my_project

# یک پوشه به بالا بروید
cd ..

# به پوشه home بروید
cd ~         # مک/لینوکس
cd %USERPROFILE%   # ویندوز
```

## اجرای اولین برنامه پایتون شما

### روش ۱: با استفاده از IDLE

1. IDLE را باز کنید
2. روی **File → New File** کلیک کنید
3. تایپ کنید: `print("سلام، دنیا!")`
4. روی **Run → Run Module** کلیک کنید (یا F5 را بزنید)
5. وقتی پرسیده شد فایل را ذخیره کنید (از پسوند `.py` استفاده کنید)

### روش ۲: با استفاده از VS Code

1. VS Code را باز کنید
2. روی **File → New File** کلیک کنید
3. آن را به عنوان `hello.py` ذخیره کنید
4. تایپ کنید: `print("سلام، دنیا!")`
5. روی دکمه "Play" (▶) در گوشه بالا سمت راست کلیک کنید، یا راست‌کلیک کرده و "Run Python File" را انتخاب کنید

### روش ۳: با استفاده از ترمینال

1. یک فایل به نام `hello.py` در ویرایشگر خود بسازید
2. این کد را اضافه کنید: `print("سلام، دنیا!")`
3. ترمینال را باز کنید
4. با استفاده از `cd` به پوشه حاوی فایل خود بروید
5. اجرا کنید: `python hello.py` (یا `python3 hello.py`)

## جریان کاری کدنویسی شما

این فرآیند معمول برای نوشتن و اجرای کد پایتون است:

```
1. ویرایشگر کد خود را باز کنید
        ↓
2. کد خود را بنویسید یا ویرایش کنید
        ↓
3. فایل را ذخیره کنید (Ctrl+S / Cmd+S)
        ↓
4. ترمینال را باز کنید
        ↓
5. به پوشه فایل خود بروید
        ↓
6. اجرا کنید: python filename.py
        ↓
7. خروجی را ببینید!
        ↓
8. برای تغییرات به مرحله ۲ برگردید
```

---

## اشتباهات رایج مبتدی‌ها

### اشتباه ۱: ذخیره نکردن قبل از اجرا

**مشکل:** تغییراتی ایجاد می‌کنید، برنامه را اجرا می‌کنید، اما خروجی قدیمی را می‌بینید.

**راه‌حل:** همیشه قبل از اجرا فایل خود را ذخیره کنید (Ctrl+S / Cmd+S).

### اشتباه ۲: پسوند فایل اشتباه

**مشکل:** فایل شما `myprogram.txt` به جای `myprogram.py` نامگذاری شده.

**راه‌حل:** فایل‌های پایتون باید با `.py` تمام شوند.

### اشتباه ۳: پایتون در PATH نیست (ویندوز)

**مشکل:** `python` را تایپ می‌کنید اما خطایی مبنی بر اینکه شناخته نشده دریافت می‌کنید.

**راه‌حل:** در حین نصب پایتون، "Add Python to PATH" را علامت بزنید. اگر این را از دست دادید، پایتون را دوباره نصب کنید.

### اشتباه ۴: اجرای پایتون از پوشه اشتباه

**مشکل:** سعی می‌کنید `python hello.py` را اجرا کنید اما در پوشه اشتباه هستید.

**راه‌حل:** با `cd` به پوشه حاوی فایل پایتون خود بروید.

## خودتان امتحان کنید: تمرین راه‌اندازی

### تمرین ۱: بررسی راه‌اندازی شما

1. ترمینال خود را باز کنید
2. `python --version` (یا `python3 --version`) را تایپ کنید
3. باید نسخه پایتون خود را ببینید
4. `python` (یا `python3`) را تایپ کنید تا وارد حالت تعاملی شوید
5. `print("کار می‌کند!")` را تایپ کنید
6. باید "کار می‌کند!" را ببینید
7. برای خروج `exit()` را تایپ کنید

### تمرین ۲: ایجاد و اجرای یک فایل

1. ویرایشگر کد خود را باز کنید
2. یک فایل جدید بسازید
3. آن را به عنوان `test.py` ذخیره کنید
4. تایپ کنید: `print("اولین برنامه پایتون من!")`
5. فایل را ذخیره کنید
6. ترمینال را باز کنید و به محل فایل بروید
7. اجرا کنید: `python test.py`
8. باید خروجی را ببینید!

## مرجع سریع

### باز کردن ترمینال

| سیستم عامل | چگونه باز کنیم |
|------------------|-------------|
| ویندوز | `Win + R` را بزنید، `cmd` را تایپ کنید |
| مک | `Cmd + Space` را بزنید، `terminal` را تایپ کنید |
| لینوکس | `Ctrl + Alt + T` را بزنید |

### اجرای پایتون

| کار | دستور |
|------|---------|
| بررسی نسخه | `python --version` |
| حالت تعاملی | `python` |
| اجرای یک فایل | `python filename.py` |

### دستورات رایج ترمینال

| کار | مک/لینوکس | ویندوز |
|------|-----------|---------|
| پوشه فعلی | `pwd` | `cd` |
| لیست فایل‌ها | `ls` | `dir` |
| تغییر پوشه | `cd foldername` | `cd foldername` |
| رفتن به بالا | `cd ..` | `cd ..` |
| رفتن به home | `cd ~` | `cd %USERPROFILE%` |

## نکات کلیدی

1. **به سه چیز نیاز دارید:** پایتون، یک ویرایشگر کد، و یک ترمینال

5. **از ترمینال استفاده کنید** برای اجرای برنامه‌های پایتون خود
6. **به پوشه درست بروید** قبل از اجرای کد خود
