# جلسه ۱۱: متغیرها و انواع داده‌های پایه (int, float, bool, str)

## مرور جلسه

حالا که می‌توانیم پایتون را اجرا کنیم، متغیرها را به عنوان مکان‌های ذخیره‌سازی نام‌گذاری شده بررسی می‌کنیم و انواع داده‌های پایه: اعداد صحیح، اعشاری، بولی‌ها و رشته‌ها.

## اهداف یادگیری کلیدی

- تعریف متغیرها در برنامه‌نویسی
- استفاده از انواع int، float، bool و str
- تخصیص مقادیر و چاپ متغیرها
- تشخیص خطاهای مربوط به نوع
