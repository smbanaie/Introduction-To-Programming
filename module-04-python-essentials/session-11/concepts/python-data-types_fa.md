# انواع داده پایتون: درک انواع مختلف اطلاعات

## چه چیزهایی یاد خواهید گرفت
- انواع داده اصلی در پایتون
- چگونه از هر نوع استفاده کنیم
- چه زمانی از کدام نوع استفاده کنیم
- چگونه بررسی کنیم چیزی از چه نوعی است

---

## انواع داده چیست؟

انواع داده دسته‌بندی‌های اطلاعات هستند. مثل دنیای واقعی که داریم:
- اعداد (برای شمارش، اندازه‌گیری)
- متن (برای نام‌ها، پیام‌ها)
- درست/نادرست (برای سوالات بله/خیر)

پایتون انواع داده مختلفی برای اهداف مختلف دارد.

---

## چهار نوع داده اصلی

### ۱. اعداد صحیح (`int`) - اعداد کامل

**آنها چه هستند:** اعداد بدون اعشار - اعداد شمارشی.

```python
# نمونه‌های اعداد صحیح
age = 25
count = 100
year = 2024
temperature = -10

print(age)        # خروجی: 25
print(count)      # خروجی: 100
```

**چه زمانی استفاده کنیم:**
- شمارش چیزها (5 سیب، 10 نفر)
- اعداد کامل (سال 2024)
- اعداد کامل منفی (دما -5)

**امتحان کنید:**
```python
my_age = 20
siblings = 2
print(my_age + siblings)   # جمع کردن اعداد صحیح
```

---

### ۲. اعداد اعشاری (`float`) - اعداد با اعشار

**آنها چه هستند:** اعداد با نقطه اعشار.

```python
# نمونه‌های اعداد اعشاری
price = 19.99
height = 1.75
pi = 3.14159
temperature = 98.6

print(price)      # خروجی: 19.99
print(height)     # خروجی: 1.75
```

**چه زمانی استفاده کنیم:**
- مقادیر پول ($19.99)
- اندازه‌گیری‌ها (1.75 متر)
- مقادیر علمی (3.14159)

**نکته مهم:**
در محاسبات پول دقت کنید! به دلیل نحوه ذخیره اعداد اعشاری در کامپیوتر:

```python
print(0.1 + 0.2)   # خروجی: 0.30000000000000004 (دقیقاً 0.3 نیست!)
```

برای محاسبات پول، بعداً ممکن است به کتابخانه‌های خاصی نیاز داشته باشید.

**امتحان کنید:**
```python
my_height = 1.70
my_weight = 65.5
print(my_height * my_weight)
```

---

### ۳. رشته‌ها (`str`) - متن

**آنها چه هستند:** دنباله‌ای از کاراکترها - متن.

```python
# نمونه‌های رشته
name = "علی"
message = 'سلام، دنیا!'
email = "ali@example.com"
phone = "555-1234"

print(name)       # خروجی: علی
print(message)    # خروجی: سلام، دنیا!
```

**گیومه تکی یا دوتایی:**
```python
# هر دو کار می‌کنند! از هر کدام که ترجیح می‌دهید استفاده کنید
name1 = "علی"
name2 = 'حسن'

# اما درون یک رشته ثابت باشید
quote = "او گفت 'سلام'"    # ترکیب گیومه‌ها OK است
quote2 = 'او گفت "سلام"'     # ترکیب گیومه‌ها OK است
```

**چه زمانی استفاده کنیم:**
- نام‌ها، آدرس‌ها، ایمیل‌ها
- پیام‌ها، توضیحات
- شماره تلفن، کدهای پستی (حتی اگر شبیه اعداد به نظر برسند!)

**چرا برای شماره تلفن از رشته استفاده کنیم؟**
```python
# ❌ اشتباه - شماره تلفن به عنوان عدد
phone = 5551234      # صفرهای ابتدایی را از دست می‌دهد!

# ✅ درست - شماره تلفن به عنوان رشته
phone = "555-1234"   # دقیقاً همانطور که تایپ شده نگه می‌دارد
```

**امتحان کنید:**
```python
first_name = "نام شما"
print("سلام، " + first_name + "!")
```

---

### ۴. بولی‌ها (`bool`) - درست یا نادرست

**آنها چه هستند:** فقط دو مقدار: `True` یا `False`.

```python
# نمونه‌های بولی
is_active = True
is_logged_in = False
has_permission = True

print(is_active)      # خروجی: True
print(is_logged_in)   # خروجی: False
```

**چه زمانی استفاده کنیم:**
- سوالات بله/خیر
- حالت‌های روشن/خاموش
- بررسی شرایط (آیا هوا گرم است؟ در باز است؟)

**امتحان کنید:**
```python
likes_python = True
print("آیا پایتون را دوست داری؟", likes_python)
```

## بررسی نوع یک مقدار

از `type()` برای دیدن اینکه چیزی از چه نوعی است استفاده کنید:

```python
print(type(42))           # <class 'int'>
print(type(3.14))         # <class 'float'>
print(type("Hello"))      # <class 'str'>
print(type(True))         # <class 'bool'>

# با متغیرها
age = 25
print(type(age))          # <class 'int'>
```

## تبدیل بین انواع

گاهی اوقات نیاز دارید مقداری را از یک نوع به نوع دیگر تغییر دهید:

### رشته به عدد

```python
# گرفتن ورودی (همیشه رشته برمی‌گرداند)
age_string = input("سن خود را وارد کنید: ")   # کاربر تایپ می‌کند: 25

# ❌ خطا - نمی‌توان با رشته‌ها ریاضی کرد
next_year = age_string + 1    # TypeError!

# ✅ ابتدا به عدد صحیح تبدیل کنید
age = int(age_string)         # "25" را به 25 تبدیل می‌کند
next_year = age + 1           # کار می‌کند! نتیجه: 26
```

### عدد به رشته

```python
age = 25

# ❌ خطا - نمی‌توان رشته و عدد را به هم چسباند
message = "من " + age + " سال دارم"    # TypeError!

# ✅ ابتدا به رشته تبدیل کنید
message = "من " + str(age) + " سال دارم"   # کار می‌کند!
print(message)   # خروجی: من 25 سال دارم

# روش بهتر: f-strings (توصیه می‌شود!)
message = f"من {age} سال دارم"
print(message)   # خروجی: من 25 سال دارم
```

### توابع تبدیل

```python
# به عدد صحیح
int("42")      # رشته "42" را به عدد صحیح 42 تبدیل می‌کند
int(3.14)      # اعشار 3.14 را به عدد صحیح 3 تبدیل می‌کند (گرد کردن به پایین)

# به اعشار
float("3.14")  # رشته "3.14" را به اعشار 3.14 تبدیل می‌کند
float(42)      # عدد صحیح 42 را به اعشار 42.0 تبدیل می‌کند

# به رشته
str(42)        # عدد صحیح 42 را به رشته "42" تبدیل می‌کند
str(3.14)      # اعشار 3.14 را به رشته "3.14" تبدیل می‌کند
str(True)      # بولی True را به رشته "True" تبدیل می‌کند
```

## اشتباهات رایج مبتدی‌ها

### اشتباه ۱: سعی در جمع کردن انواع مختلف

```python
# ❌ خطا: نمی‌توان رشته و عدد را جمع کرد
result = "5" + 3    # TypeError!

# ✅ رفع: یکی از آنها را تبدیل کنید
result = int("5") + 3     # نتیجه: 8
result = "5" + str(3)     # نتیجه: "53" (الحاق)
```

### اشتباه ۲: فراموش کردن اینکه input رشته برمی‌گرداند

```python
# ❌ اشتباه رایج
age = input("سن خود را وارد کنید: ")
next_year = age + 1    # TypeError!

# ✅ درست
age = int(input("سن خود را وارد کنید: "))
next_year = age + 1    # کار می‌کند!
```

### اشتباه ۳: تبدیل رشته‌های نامعتبر به اعداد

```python
# ❌ خطا: نمی‌توان این متن را به عدد تبدیل کرد
number = int("سلام")    # ValueError!

# ✅ ابتدا بررسی کنید یا از try-except استفاده کنید
user_input = input("یک عدد وارد کنید: ")
try:
    number = int(user_input)
    print(f"شما وارد کردید: {number}")
except ValueError:
    print("این یک عدد معتبر نیست!")
```

## خودتان امتحان کنید: تمرین‌ها

### تمرین ۱: کاراگاه نوع

هر کدام از اینها از چه نوعی هستند؟ حدس بزنید، سپس با `type()` بررسی کنید.

```python
print(type(100))
print(type("100"))
print(type(100.0))
print(type(True))
print(type("True"))
```

**پاسخ‌ها:**
- `100` → int
- `"100"` → str
- `100.0` → float
- `True` → bool
- `"True"` → str (در گیومه متن است!)

### تمرین ۲: مبدل دما

تبدیل دما از سلسیوس به فارنهایت.

فرمول: `F = C × 9/5 + 32`

```python
# گرفتن دما از کاربر
celsius = float(input("دما به سلسیوس را وارد کنید: "))

# محاسبه فارنهایت
fahrenheit = celsius * 9/5 + 32

# نمایش نتیجه
print(f"{celsius}°C برابر است با {fahrenheit}°F")
```

### تمرین ۳: تبدیل رشته به عدد

این کد را برای کار کردن درست کنید:

```python
# اصلی (خراب)
num1 = input("عدد اول را وارد کنید: ")
num2 = input("عدد دوم را وارد کنید: ")
sum = num1 + num2
print(f"مجموع: {sum}")

# وقتی 5 و 3 را وارد می‌کنید، "53" را چاپ می‌کند به جای 8!
# آن را طوری درست کنید که اعداد را درست جمع کند.
```

### تمرین ۴: ساخت یک پروفایل

یک برنامه بسازید که بپرسد و ذخیره کند:
- نام (رشته)
- سن (عدد صحیح)
- قد به متر (اعشار)
- آیا دانشجو است (بولی)

سپس همه اطلاعات را نمایش دهد.

```python
# کد شما اینجا
name = input("نام خود را وارد کنید: ")
age = int(input("سن خود را وارد کنید: "))
height = float(input("قد خود را به متر وارد کنید: "))
is_student = input("آیا دانشجو هستید؟ (بله/خیر): ").lower() == "بله"

print("\n--- پروفایل شما ---")
print(f"نام: {name}")
print(f"سن: {age}")
print(f"قد: {height}m")
print(f"دانشجو: {is_student}")
```

## مرجع سریع

| نوع | مثال | استفاده برای |
|------|---------|---------|
| `int` | `42`, `-5` | اعداد کامل، شمارش |
| `float` | `3.14`, `19.99` | اعشار، اندازه‌گیری، پول |
| `str` | `"سلام"`, `'دنیا'` | متن، نام، پیام |
| `bool` | `True`, `False` | بله/خیر، روشن/خاموش |

### توابع تبدیل

| تابع | تبدیل به | مثال |
|----------|-------------|---------|
| `int()` | عدد صحیح | `int("42")` → `42` |
| `float()` | اعشار | `float("3.14")` → `3.14` |
| `str()` | رشته | `str(42)` → `"42"` |

---

## نکات کلیدی

1. **چهار نوع اصلی:** `int` (اعداد کامل)، `float` (اعشار)، `str` (متن)، `bool` (درست/نادرست)
2. **نوع مناسب را انتخاب کنید** برای داده‌های خود
3. **input همیشه رشته برمی‌گرداند** - وقتی نیاز است به عدد تبدیل کنید
4. **از `type()` برای بررسی استفاده کنید** چیزی از چه نوعی است
5. **از `int()`, `float()`, `str()`** برای تبدیل بین انواع استفاده کنید
6. **f-strings فرمت‌بندی را آسان می‌کند** - `f"من {age} سال دارم"`
