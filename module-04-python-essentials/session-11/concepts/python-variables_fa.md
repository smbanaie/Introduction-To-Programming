## متغیر چیست؟

یک **متغیر** مثل یک **ظرف برچسب‌دار** است که اطلاعات مورد نیاز برنامه شما را ذخیره می‌کند.

آن را مثل یک جعبه با برچسب نام در نظر بگیرید:

```
┌─────────────────┐
│   جعبه: "age"   │  ← برچسب (نام متغیر)
│                 │
│    محتوا:       │
│      25         │  ← مقدار ذخیره شده
└─────────────────┘
```

### تشبیه دنیای واقعی

فرض کنید دارید اتاق خود را مرتب می‌کنید:
- یک جعبه دارید با برچسب "جوراب" که حاوی جوراب‌های شماست
- یک جعبه دارید با برچسب "کتاب" که حاوی کتاب‌های شماست
- یک جعبه دارید با برچسب "اسباب‌بازی" که حاوی اسباب‌بازی‌های شماست

در پایتون:
- `socks = "wool socks"` یک "جعبه" به نام socks می‌سازد
- `books = 15` یک "جعبه" به نام books با عدد 15 می‌سازد

## ساختن اولین متغیرهای شما

ساختن یک متغیر ساده است: یک نام به آن بدهید، از `=` (عملگر انتساب) استفاده کنید، و یک مقدار به آن بدهید.

```python
# ساختن متغیرها
name = "علی"
age = 25
height = 1.65
is_student = True

# استفاده از متغیرها
print(name)        # خروجی: علی
print(age)         # خروجی: 25
print(height)      # خروجی: 1.65
print(is_student)  # خروجی: True
```

### عملگر انتساب (=)

علامت `=` در پایتون به معنای "برابر است" مثل ریاضی نیست. به معنای **"مقدار سمت راست را به نام سمت چپ بده"** است.

```python
# این را بخوانید: "x مقدار 10 را می‌گیرد"
x = 10

# این را بخوانید: "name مقدار 'حسن' را می‌گیرد"
name = "حسن"
```

## قوانین نام‌گذاری متغیرها

پایتون قوانینی برای نام‌گذاری متغیرها دارد:

### ✅ نام‌های معتبر

```python
MAX_SIZE = 50          # همه حروف بزرگ مجاز است (قرارداد برای ثابت‌ها)
```

### ❌ نام‌های نامعتبر (باعث خطا می‌شوند)

```python
# اینها کار نخواهند کرد:
2name = "علی"          # نمی‌تواند با عدد شروع شود
my-name = "حسن"        # نمی‌تواند از خط تیره استفاده کند
```

```python
# از اینها به عنوان نام متغیر استفاده نکنید:
```

## بهترین روش‌ها برای نام‌گذاری متغیرها

### از نام‌های توصیفی استفاده کنید

**❌ بد - نامشخص است متغیر چه معنایی دارد:**
```python
x = 25
n = "علی"
d = 3.14
```

**✅ خوب - روشن و توصیفی:**
```python
age = 25
username = "علی"
pi_value = 3.14
```

### از snake_case استفاده کنید (حروف کوچک با زیرخط)

قرارداد پایتون استفاده از حروف کوچک با زیرخط بین کلمات است:

```python
# ✅ خوب (snake_case)
user_name = "علی"
total_score = 100
is_valid = True

# ❌ از این سبک‌ها در پایتون اجتناب کنید
userName = "علی"       # camelCase (در زبان‌های دیگر رایج است)
UserName = "علی"       # PascalCase (برای کلاس‌ها استفاده می‌شود)
username = "علی"       # حروف کوچک (برای کلمات تک OK است)
```

## به‌روزرسانی مقادیر متغیرها

متغیرها می‌توانند مقادیرشان را تغییر دهند. به همین دلیل به آنها "متغیر" گفته می‌شود - می‌توانند متغیر باشند!

### به‌روزرسانی‌های ساده

```python
score = 0
print(score)      # خروجی: 0

score = 10
print(score)      # خروجی: 10

score = 25
print(score)      # خروجی: 25
```

### استفاده از مقدار فعلی یک متغیر

```python
counter = 5
print(counter)    # خروجی: 5

# استفاده از مقدار فعلی و اضافه کردن 1
counter = counter + 1
print(counter)    # خروجی: 6

# نسخه خلاصه (همان کار را می‌کند)
counter += 1      # اضافه کردن 1 به counter
counter += 5      # اضافه کردن 5 به counter
counter -= 2      # کم کردن 2 از counter
```

### عملگرهای خلاصه رایج

```python
x = 10

x += 5      # مثل: x = x + 5      (x الان 15)
x -= 3      # مثل: x = x - 3      (x الان 12)
x *= 2      # مثل: x = x * 2      (x الان 24)
x /= 4      # مثل: x = x / 4      (x الان 6.0)
```

## اشتباهات رایج مبتدی‌ها

### اشتباه ۱: استفاده از متغیرهای تعریف‌نشده

```python
# ❌ خطا: NameError
print(my_variable)   # خطا! متغیر هنوز وجود ندارد

# ✅ رفع: ابتدا متغیر را بسازید
my_variable = "سلام"
print(my_variable)   # کار می‌کند!
```

### اشتباه ۲: اشتباه گرفتن = با ==

```python
# ❌ اشتباه: = برای انتساب است، نه مقایسه
if x = 5:
    print("x برابر 5 است")

# ✅ درست: == برای مقایسه است
if x == 5:
    print("x برابر 5 است")
```

### اشتباه ۳: غلط املایی در نام متغیرها

```python
name = "علی"

# ❌ خطا: NameError (املای متفاوت)
print(naem)   # خطا! 'naem' تعریف نشده

# ✅ درست
print(name)   # کار می‌کند!
```

### اشتباه ۴: سعی در استفاده از کلمات رزرو شده

```python
# ❌ خطا: SyntaxError
class = "پایتون 101"   # 'class' کلمه رزرو شده است!

# ✅ رفع: از نام متفاوت استفاده کنید
course_name = "پایتون 101"
```

## خودتان امتحان کنید: تمرین‌ها

### تمرین ۱: متغیرهای خودتان را بسازید

متغیرهایی برای این موارد بسازید:
- نام شما
- سن شما
- عدد مورد علاقه شما
- آیا پیتزا دوست دارید (True یا False)

سپس هر کدام را چاپ کنید!

```python
name = "نام شما"
age = 20
favorite_number = 7
likes_pizza = True

print(name)
print(age)
print(favorite_number)
print(likes_pizza)
```

### تمرین ۲: چالش به‌روزرسانی متغیر

با `score = 0` شروع کنید. آن را به‌روز کنید تا:
1. اضافه کردن 10
2. اضافه کردن 25
3. کم کردن 5
4. دو برابر کردن (ضرب در 2)

امتیاز نهایی چیست؟

```python
score = 0
score += 10
score += 25
score -= 5
score *= 2
print(score)   # این چه چیزی چاپ خواهد کرد؟
```

### تمرین ۳: رفع خطاها

خطاها در این کد را پیدا و رفع کنید:

```python
# اصلی (با خطاها)
2nd_place = "حسن"
my-score = 100
my name = "علی"
class = "پایتون"

```

### تمرین ۴: جابه‌جایی مقادیر

آیا می‌توانید مقادیر دو متغیر را بدون از دست دادن هیچ‌کدام جابه‌جا کنید؟

```python
a = 5
b = 10

# کد شما اینجا برای جابه‌جایی a و b
# طوری که a شود 10 و b شود 5

print(a)  # باید چاپ کند: 10
print(b)  # باید چاپ کند: 5
```

**راهنمایی:** ممکن است به یک متغیر موقت نیاز داشته باشید، یا از ویژگی جالب چندتایی انتساب پایتون استفاده کنید!

<details>
<summary>برای دیدن دو راه‌حل کلیک کنید</summary>

**راه‌حل ۱ - با استفاده از متغیر موقت:**
```python
a = 5
b = 10
temp = a
a = b
b = temp
print(a, b)  # 10 5
```

**راه‌حل ۲ - روش جالب پایتون:**
```python
a = 5
b = 10
a, b = b, a   # جادوی پایتون!
print(a, b)   # 10 5
```
</details>

## مرجع سریع

| مفهوم | مثال | معنی |
|---------|---------|---------|
| ساخت متغیر | `x = 5` | ذخیره 5 در ظرفی به نام x |
| به‌روزرسانی متغیر | `x = 10` | تغییر x به 10 |
| اضافه کردن به متغیر | `x += 3` | اضافه کردن 3 به مقدار فعلی x |
| نام معتبر | `my_var` | حروف، اعداد، زیرخط |
| نام نامعتبر | `2var`, `my-var` | نمی‌تواند با عدد شروع شود، بدون خط تیره |

---

## نکات کلیدی

1. **متغیرها اطلاعات را ذخیره می‌کنند** - مثل جعبه‌های برچسب‌دار
2. **با `=` بسازید** - name = value
3. **نام‌ها باید توصیفی باشند** - به جای `x` از `user_age` استفاده کنید
4. **از قوانین نام‌گذاری پیروی کنید** - بدون فاصله، بدون شروع با عدد
5. **از snake_case استفاده کنید** - حروف کوچک با زیرخط
6. **متغیرها می‌توانند تغییر کنند** - به همین دلیل به آنها "متغیر" گفته می‌شود
