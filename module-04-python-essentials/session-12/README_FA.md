# جلسه ۱۲: ورودی/خروجی و عبارت‌ها و عملگرها

## مرور جلسه

تعامل کاربر را با input() و قالب‌بندی خروجی اضافه می‌کنیم، بعلاوه عبارت‌هایی با استفاده از عملگرهای حسابی، مقایسه و منطقی می‌سازیم.

## اهداف یادگیری کلیدی

- استفاده از input() برای خواندن داده‌های کاربر
- تبدیل ورودی‌ها به انواع مناسب
- ساخت عبارت‌ها با عملگرها
- نوشتن برنامه‌هایی که I/O و پردازش را ترکیب می‌کنند
