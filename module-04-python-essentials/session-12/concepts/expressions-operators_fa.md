# عبارات و عملگرها: ماشین‌حساب پایتون

## چه چیزهایی یاد خواهید گرفت
- چگونه در پایتون حساب کنیم (عملگرهای حسابی)
- چگونه مقادیر را مقایسه کنیم (عملگرهای مقایسه)
- چگونه شرایط را ترکیب کنیم (عملگرهای منطقی)
- ترتیب حل مسائل ریاضی در پایتون (تقدم عملگر)
- اشتباهات رایج مبتدی‌ها

---

## مفهوم اصلی: پایتون به عنوان ماشین‌حساب

پایتون را مثل یک ماشین‌حساب فوق‌العاده قدرتمند در نظر بگیرید. می‌توانید **عبارات** بنویسید—ترکیبی از اعداد، متغیرها و عملگرها—که پایتون آن‌ها را محاسبه کرده و نتیجه را به شما بدهد.

**مثال: پیروی از یک دستور پخت**
مثل دستور پخت که می‌گوید ابتدا مواد را مخلوط کنید (قبل از پخت)، پایتون هم قوانینی دارد که کدام عملیات اول انجام شود.

## عملگرهای حسابی: ریاضی پایه

### چهار عمل اصلی

```python
# جمع (+)
سیب = 5
پرتقال = 3
کل_میوه = سیب + پرتقال
print(f"کل میوه: {کل_میوه}")  # 8

# تفریق (-)
پول = 100
خرج = 35
باقیمانده = پول - خرج
print(f"باقیمانده: {باقیمانده}")  # 65

# ضرب (*)
جعبه = 4
تعداد_در_جعبه = 6
کل_اقلام = جعبه * تعداد_در_جعبه
print(f"کل اقلام: {کل_اقلام}")  # 24

# تقسیم (/)
کل_کوکی = 12
افراد = 4
کوکی_هر_نفر = کل_کوکی / افراد
print(f"کوکی برای هر نفر: {کوکی_هر_نفر}")  # 3.0
```

### عملگرهای تقسیم ویژه

```python
# تقسیم صحیح (//) - اعشار را حذف می‌کند
پیتزا = 10
افراد = 3
پیتزای_کامل_هر_نفر = پیتزا // افراد
print(f"پیتزای کامل برای هر نفر: {پیتزای_کامل_هر_نفر}")  # 3

# باقیمانده (%) - باقیمانده تقسیم را می‌دهد
اسلایس_باقیمانده = پیتزا % افراد
print(f"اسلایس باقیمانده: {اسلایس_باقیمانده}")  # 1

# توان (**) - به توان رساندن
مربع = 5 ** 2  # 5 به توان 2 = 25
مکعب = 2 ** 3  # 2 به توان 3 = 8
print(f"مربع 5: {مربع}، مکعب 2: {مکعب}")
```

### مرجع سریع: عملگرهای ریاضی

| عملگر | نام | مثال | نتیجه |
|--------|------|--------|--------|
| `+` | جمع | `5 + 3` | `8` |
| `-` | تفریق | `10 - 4` | `6` |
| `*` | ضرب | `6 * 7` | `42` |
| `/` | تقسیم | `15 / 4` | `3.75` |
| `//` | تقسیم صحیح | `15 // 4` | `3` |
| `%` | باقیمانده | `15 % 4` | `3` |
| `**` | توان | `2 ** 3` | `8` |

## عملگرهای مقایسه: مقایسه مقادیر

### مقایسه‌های پایه

```python
سن = 25

# برابر با (==)
بیست_و_پنج = (سن == 25)  # True

# نابرابر با (!=)
سی_نیست = (سن != 30)  # True

# بزرگ‌تر از (>)
بزرگسال = (سن > 18)  # True

# کوچک‌تر از (<)
کودک = (سن < 13)  # False

# بزرگ‌تر یا مساوی (>=)
می‌تواند_رای_دهد = (سن >= 18)  # True

# کوچک‌تر یا مساوی (<=)
تخفیف_دارد = (سن <= 12)  # False

print(f"می‌تواند رای دهد: {می‌تواند_رای_دهد}")
```

### مقایسه متن (رشته‌ها)

```python
نام = "علی"

# بررسی برابری (به بزرگی و کوچکی حروف حساس!)
علی_هست = (نام == "علی")  # True
علی_کوچک_هست = (نام == "ali")  # False (حروف مهم هستند!)

# بررسی خالی نبودن
نام_دارد = bool(نام)  # True (رشته غیرخالی)
خالی = ""
خالی_هست = bool(خالی)  # False
```

## عملگرهای منطقی: ترکیب شرایط

### AND، OR و NOT

عملگرهای منطقی را مثل سوالاتی که می‌پرسید در نظر بگیرید:

```python
بلیط_دارد = True
بزرگسال_هست = True

# AND - هر دو باید درست باشند (مثل "آیا بلیط داری AND بزرگسال هستی؟")
می‌تواند_وارد_شود = بلیط_دارد and بزرگسال_هست
print(f"می‌تواند وارد شود: {می‌تواند_وارد_شود}")  # True

# OR - حداقل یکی باید درست باشد (مثل "آیا بلیط داری OR عضو هستی؟")
عضو_هست = False
می‌تواند_وارد_شود_۲ = بلیط_دارد or عضو_هست  # True (بلیط دارد)

# NOT - معکوس می‌کند
باران_می‌بارد = False
باید_بیرون_رفت = not باران_می‌بارد  # True (باران نمی‌بارد)
```

### مثال‌های واقعی

```python
# بررسی اینکه آیا کسی می‌تواند رانندگی کند
سن = 20
گواهینامه_دارد = True
هوشیار_هست = True

می‌تواند_رانندگی_کند = سن >= 18 and گواهینامه_دارد and هوشیار_هست
print(f"می‌تواند رانندگی کند: {می‌تواند_رانندگی_کند}")  # True

# بررسی آخر هفته
روز = "شنبه"
آخر_هفته_هست = (روز == "شنبه") or (روز == "یکشنبه")
print(f"آخر هفته هست: {آخر_هفته_هست}")  # True
```

### جدول درستی (مرجع سریع)

| شرط ۱ | شرط ۲ | نتیجه AND | نتیجه OR |
|--------|--------|-----------|----------|
| True | True | True | True |
| True | False | False | True |
| False | True | False | True |
| False | False | False | False |

## تقدم عملگر: ترتیب عملیات

### پایتون از PEMDAS پیروی می‌کند (مثل کلاس ریاضی!)

مثل کلاس ریاضی، پایتون عملیات را به ترتیب خاصی انجام می‌دهد:

۱. **P**erantez `()` - همیشه اول!
۲. **E**xponent (توان) `**` - توان و ریشه
۳. **M**ultiply (ضرب) `*` و **D**ivide (تقسیم) `/` (چپ به راست)
۴. **A**dd (جمع) `+` و **S**ubtract (تفریق) `-` (چپ به راست)
۵. **C**ompare (مقایسه) `==`، `>`، `<` و غیره
۶. **L**ogical (منطقی) `not`، `and`، `or`

```python
# بدون پرانتز
نتیجه = 2 + 3 * 4
print(نتیجه)  # 14 (ضرب اول: 3*4=12، بعد 2+12)

# با پرانتز
نتیجه = (2 + 3) * 4
print(نتیجه)  # 20 (پرانتز اول: 2+3=5، بعد 5*4)

# مثال پیچیده‌تر
مجموع = 10 + 5 * 2 ** 2
# مرحله ۱: 2 ** 2 = 4
# مرحله ۲: 5 * 4 = 20
# مرحله ۳: 10 + 20 = 30
print(مجموع)  # 30
```

### وقتی شک داشتید، از پرانتز استفاده کنید!

```python
# واضح و ایمن
نتیجه = (قیمت * تعداد) + (حمل_ونقل * 0.1)

# گیج‌کننده
نتیجه = قیمت * تعداد + حمل_ونقل * 0.1
# (نتیجه یکسان، اما خواندن سخت‌تر)
```

## اشتباهات رایج مبتدی‌ها

### اشتباه ۱: استفاده از `=` به جای `==`

```python
# ❌ اشتباه - این تلاش برای مقداردهی است، نه مقایسه
if سن = 18:  # SyntaxError!
    print("۱۸ سال داری")

# ✅ درست - مقایسه مقادیر
if سن == 18:
    print("دقیقاً ۱۸ سال داری")
```

### اشتباه ۲: مقایسه رشته با عدد

```python
# ❌ اشتباه - مقایسه رشته با عدد
سن = "25"  # رشته از ورودی()
if سن == 25:  # هیچ‌وقت درست نیست!
    print("بزرگسال")

# ✅ درست - ابتدا تبدیل کن
سن = int("25")
if سن == 25:
    print("بزرگسال")
```

### اشتباه ۳: فراموش کردن تقدم عملگر

```python
# ❌ اشتباه - ممکن است آن چیزی که انتظار دارید نباشد
نتیجه = 10 / 2 * 5  # 25.0 (نه 1.0)
# تقسیم قبل از ضرب انجام می‌شود (چپ به راست)

# ✅ درست - از پرانتز برای وضوح استفاده کنید
tخفیف = مجموع * (درصد / 100)
```

### اشتباه ۴: اشتباه گرفتن `/` و `//`

```python
# تقسیم معمولی اعشار می‌دهد
نتیجه۱ = 10 / 3  # 3.333...

# تقسیم صحیح عدد صحیح می‌دهد
نتیجه۲ = 10 // 3  # 3

# وقتی به مقادیر دقیق نیاز دارید مراقب باشید!
```

### اشتباه ۵: باقیمانده با اعداد منفی

```python
# باقیمانده با اعداد منفی می‌تواند گیج‌کننده باشد
# به عنوان مبتدی با اعداد مثبت کار کنید
باقیمانده = 17 % 5  # 2 (قابل درک)
```

## خودتان امتحان کنید: تمرین‌ها

### تمرین ۱: ماشین‌حساب ساده

برنامه‌ای برای محاسبه مساحت مستطیل بسازید.

```python
# گرفتن ابعاد
عرض = float(input("عرض را وارد کن: "))
ارتفاع = float(input("ارتفاع را وارد کن: "))

# محاسبه مساحت و محیط
مساحت = عرض * ارتفاع
محیط = 2 * (عرض + ارتفاع)

# نمایش نتایج
print(f"مساحت: {مساحت}")
print(f"محیط: {محیط}")
```

### تمرین ۲: مبدل دما

سلسیوس را به فارنهایت تبدیل کنید.

```python
# فرمول: F = (C × 9/5) + 32
سلسیوس = float(input("دما به سلسیوس وارد کن: "))
فارنهایت = (سلسیوس * 9/5) + 32
print(f"{سلسیوس}°C = {فارنهایت}°F")
```

### تمرین ۳: ماشین‌حساب خرید

مجموع را با مالیات و تخفیف محاسبه کنید.

```python
قیمت = float(input("قیمت کالا: $"))
تعداد = int(input("تعداد: "))
نرخ_مالیات = 0.08  # ۸٪ مالیات
حد_تخفیف = 100  # ۱۰۰ دلار برای تخفیف
نرخ_تخفیف = 0.10  # ۱۰٪ تخفیف

جمع_جزی = قیمت * تعداد

# اعمال تخفیف اگر از حد بیشتر باشد
if جمع_جزی > حد_تخفیف:
    تخفیف = جمع_جزی * نرخ_تخفیف
    جمع_جزی = جمع_جزی - تخفیف
    print(f"تخفیف اعمال شد: ${تخفیف:.2f}")

مالیات = جمع_جزی * نرخ_مالیات
مجموع = جمع_جزی + مالیات

print(f"جمع جزئی: ${جمع_جزی:.2f}")
print(f"مالیات: ${مالیات:.2f}")
print(f"مجموع: ${مجموع:.2f}")
```

## مرجع سریع

| دسته | عملگر | کار | مثال |
|------|--------|-----|------|
| **ریاضی** | `+` | جمع | `5 + 3 = 8` |
| **ریاضی** | `-` | تفریق | `10 - 4 = 6` |
| **ریاضی** | `*` | ضرب | `6 * 7 = 42` |
| **ریاضی** | `/` | تقسیم | `15 / 4 = 3.75` |
| **ریاضی** | `//` | تقسیم صحیح | `15 // 4 = 3` |
| **ریاضی** | `%` | باقیمانده | `15 % 4 = 3` |
| **ریاضی** | `**` | توان | `2 ** 3 = 8` |
| **مقایسه** | `==` | برابر با | `5 == 5` → `True` |
| **مقایسه** | `!=` | نابرابر با | `5 != 3` → `True` |
| **مقایسه** | `>` | بزرگ‌تر از | `7 > 5` → `True` |
| **مقایسه** | `<` | کوچک‌تر از | `3 < 8` → `True` |
| **مقایسه** | `>=` | بزرگ‌تر یا مساوی | `5 >= 5` → `True` |
| **مقایسه** | `<=` | کوچک‌تر یا مساوی | `4 <= 6` → `True` |
| **منطقی** | `and` | هر دو درست | `True and True` → `True` |
| **منطقی** | `or` | حداقل یکی درست | `True or False` → `True` |
| **منطقی** | `not` | معکوس | `not True` → `False` |

### ترتیب تقدم (از بالاتر به پایین‌تر)

۱. `()` - پرانتز
۲. `**` - توان
۳. `*`، `/`، `//`، `%` - ضرب/تقسیم
۴. `+`، `-` - جمع/تفریق
۵. `==`، `!=`، `>`، `<`، `>=`، `<=` - مقایسه
۶. `not` - منطقی NOT
۷. `and` - منطقی AND
۸. `or` - منطقی OR

---

## نکات کلیدی

۱. **عملگرهای حسابی** (`+`، `-`، `*`، `/`، `//`، `%`، `**`) محاسبات ریاضی انجام می‌دهند
۲. **عملگرهای مقایسه** (`==`، `!=`، `>`، `<`، `>=`، `<=`) مقادیر را مقایسه کرده و True/False برمی‌گردانند
۳. **عملگرهای منطقی** (`and`، `or`، `not`) شرایط متعدد را ترکیب می‌کنند
۴. **تقدم عملگر** تعیین می‌کند کدام عملیات اول انجام شود—برای وضوح از پرانتز استفاده کنید
۵. **همیشه از `==`** برای مقایسه استفاده کنید، نه `=` (که برای مقداردهی است)
۶. **ورودی را تبدیل کنید** به عدد با `int()` یا `float()` قبل از انجام محاسبات

## گام بعدی

حالا که می‌دانید چگونه محاسبات و مقایسه‌ها را انجام دهید، یاد می‌گیرید:
- چگونه پایتون تصمیم می‌گیرد کدام عملیات اول انجام شود (ترتیب ارزیابی)
- چگونه تصمیم بگیرید در برنامه (دستورات if/else)
- چگونه عملیات را تکرار کنید (حلقه‌ها)
- چگونه با مجموعه‌های داده کار کنید (لیست‌ها)
