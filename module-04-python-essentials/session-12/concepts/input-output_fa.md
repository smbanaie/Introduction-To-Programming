# ورودی و خروجی: صحبت با برنامه شما

## چه چیزهایی یاد خواهید گرفت
- چگونه اطلاعات را از کاربران بگیریم (ورودی)
- چگونه اطلاعات را به کاربران نشان دهیم (خروجی)
- چگونه متن را زیبا قالب‌بندی کنیم
- اشتباهات رایج مبتدی‌ها

---

## ورودی و خروجی چیست؟

برنامه شما را مثل یک مکالمه در نظر بگیرید:
- **ورودی** = کاربر با برنامه صحبت می‌کند (تایپ اطلاعات)
- **خروجی** = برنامه با کاربر صحبت می‌کند (نمایش متن)

```
کاربر: "اسمت چیه؟"  ← این چیزی است که کاربر تایپ می‌کند (ورودی)
برنامه: "سلام، علی!"  ← این چیزی است که برنامه نشان می‌دهد (خروجی)
```

---

## خروجی پایه: تابع print()

تابع `print()` نحوه صحبت کردن برنامه با کاربر است.

### چاپ ساده

```python
print("سلام، دنیا!")
print("من دارم پایتون یاد می‌گیرم")
```

**خروجی:**
```
سلام، دنیا!
من دارم پایتون یاد می‌گیرم
```

### چاپ چند چیز

```python
name = "علی"
age = 25

# چاپ چند مورد
print("نام:", name)
print("سن:", age)
print("نام:", name, "سن:", age)
```

**خروجی:**
```
نام: علی
سن: 25
نام: علی سن: 25
```

### سفارشی‌سازی print()

```python
# در همان خط بمان (بدون خط جدید)
print("سلام", end=" ")
print("دنیا")  # خروجی: سلام دنیا

# از جداکننده سفارشی استفاده کن
print("سیب", "موز", "گیلاس", sep="، ")
# خروجی: سیب، موز، گیلاس
```

## ورودی پایه: تابع input()

تابع `input()` به برنامه شما اجازه می‌دهد سوال بپرسد و جواب بگیرد.

### ورودی ساده

```python
name = input("اسمت چیه؟ ")
print("سلام، " + name + "!")
```

**نحوه کار:**
1. برنامه نشان می‌دهد: "اسمت چیه؟ "
2. کاربر تایپ می‌کند: علی
3. برنامه "علی" را در متغیر `name` ذخیره می‌کند
4. برنامه چاپ می‌کند: "سلام، علی!"

### ورودی همیشه متن برمی‌گرداند (رشته)

```python
age = input("چند سال داری؟ ")
print(type(age))  # نشان می‌دهد: <class 'str'> (متن، نه عدد!)
```

**مهم:** حتی اگر کاربر عدد تایپ کند، `input()` متن به شما می‌دهد.

### تبدیل ورودی به عدد

```python
# گرفتن عدد صحیح
age = int(input("چند سال داری؟ "))
print("سال بعدی", age + 1, "سالته")

# گرفتن عدد اعشاری
height = float(input("قدت به متر: "))
print("قدت", height, "متر است")
```

| تابع | استفاده برای | مثال |
|------|-------------|------|
| `int()` | اعداد صحیح | `int("25")` → `25` |
| `float()` | اعداد اعشاری | `float("1.75")` → `1.75` |
| `str()` | تبدیل به متن | `str(25)` → `"25"` |

## قالب‌بندی رشته: زیباسازی خروجی

### f-strings (توصیه شده)

ساده‌ترین روش قالب‌بندی متن در پایتون ۳.۶+:

```python
name = "علی"
age = 25
height = 1.65

# متغیرها را داخل آکولاد بگذار
print(f"سلام، {name}!")
print(f"تو {age} سال داری")
print(f"قدت {height} متر است")

# محاسبات داخل آکولاد
print(f"سال بعدی {age + 1} سالته")
print(f"دو برابر سنت {age * 2} است")
```

**خروجی:**
```
سلام، علی!
تو 25 سال داری
قدت 1.65 متر است
سال بعدی 26 سالته
دو برابر سنت 50 است
```

### قالب‌بندی اعداد

```python
pi = 3.14159265359
price = 19.99

# گرد کردن به ۲ رقم اعشار
print(f"پی تقریباً {pi:.2f} است")  # 3.14

# نمایش قیمت
print(f"قیمت: {price:.2f} تومان")  # 19.99 تومان

# اضافه کردن کاما برای اعداد بزرگ
population = 80000000
print(f"جمعیت ایران: {population:,} نفر")  # 80,000,000 نفر
```

## اشتباهات رایج مبتدی‌ها

### اشتباه ۱: فراموش کردن تبدیل ورودی

```python
# ❌ اشتباه - تلاش برای جمع متن با عدد
age = input("چند سال داری؟ ")
next_year = age + 1  # خطا! نمی‌توان متن + عدد کرد

# ✅ درست
age = int(input("چند سال داری؟ "))
next_year = age + 1  # کار می‌کند!
```

### اشتباه ۲: گیومه‌های ناسازگار

```python
# ❌ اشتباه - ترکیب انواع گیومه
print("سلام')

# ✅ درست
print("سلام")
print('سلام')
```

### اشتباه ۳: فراموش کردن پیشوند f-string

```python
name = "علی"

# ❌ اشتباه - نبود 'f'
print("سلام، {name}!")  # نشان می‌دهد: سلام، {name}!

# ✅ درست
print(f"سلام، {name}!")  # نشان می‌دهد: سلام، علی!
```

### اشتباه ۴: فاصله در ورودی

```python
# فاصله بعد از سوال خروجی را بهتر می‌کند
name = input("اسمت چیه؟")   # کاربر کنار ؟ تایپ می‌کند
name = input("اسمت چیه؟ ")  # بهتر - فاصله قبل از ورودی
```

### اشتباه ۵: NameError - استفاده از نام متغیر اشتباه

```python
# ❌ اشتباه - 'Age' با 'age' فرق دارد
Age = input("چند سال داری؟ ")
print(f"تو {age} سال داری")  # خطا - 'age' تعریف نشده

# ✅ درست
age = input("چند سال داری؟ ")
print(f"تو {age} سال داری")
```

## خودتان امتحان کنید: تمرین‌ها

### تمرین ۱: سلام شخصی

برنامه‌ای بسازید که نام و سن کاربر را بپرسد، بعد یک سلام چاپ کند.

```python
name = input("اسمت چیه؟ ")
age = int(input("چند سال داری؟ "))
print(f"سلام، {name}! تو {age} سال داری.")
```

### تمرین ۲: ماشین‌حساب ساده

برنامه‌ای بسازید که دو عدد را با هم جمع کند.

```python
print("=== ماشین‌حساب جمع ساده ===")
num1 = float(input("عدد اول را وارد کن: "))
num2 = float(input("عدد دوم را وارد کن: "))
total = num1 + num2
print(f"{num1} + {num2} = {total}")
```

### تمرین ۳: ماشین‌حساب مستطیل

مساحت و محیط یک مستطیل را حساب کن.

```python
print("=== ماشین‌حساب مستطیل ===")
width = float(input("عرض را وارد کن: "))
height = float(input("ارتفاع را وارد کن: "))

area = width * height
perimeter = 2 * (width + height)

print(f"مساحت: {area}")
print(f"محیط: {perimeter}")
```

### تمرین ۴: رفع اشکالات

این برنامه ۴ خطا دارد. آیا می‌توانید آنها را پیدا و رفع کنید؟

```python
# برنامه با اشکال
name = input("اسمت چیه؟ )
Age = int(input("چند سال داری؟ "))
print("سلام " + name + ، تو", age, "سال داری")
```

## مرجع سریع

| کار | کد نمونه |
|-----|---------|
| چاپ متن | `print("سلام")` |
| گرفتن ورودی متن | `name = input("نام؟ ")` |
| گرفتن ورودی عدد | `age = int(input("سن؟ "))` |
| قالب‌بندی با f-string | `print(f"سلام، {name}")` |
| قالب‌بندی عدد | `print(f"{price:.2f}")` |
| تبدیل به متن | `str(42)` |
| تبدیل به عدد صحیح | `int("42")` |
| تبدیل به عدد اعشاری | `float("3.14")` |

---

## نکات کلیدی

1. **`print()`** خروجی را به کاربر نشان می‌دهد
2. **`input()`** متن را از کاربر می‌گیرد
3. **`input()` همیشه متن برمی‌گرداند** - برای اعداد از `int()` یا `float()` استفاده کنید
4. **f-strings** (`f"سلام، {name}"`) قالب‌بندی را آسان می‌کنند
5. **متغیرها مقادیر را ذخیره می‌کنند** برای استفاده بعدی
