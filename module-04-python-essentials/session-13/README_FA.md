# جلسه ۱۳: تصمیم‌گیری با if, elif, else

## مرور جلسه

امروز اجرای شرطی را با دستورات if، elif و else یاد می‌گیریم، تبدیل شبه‌کد تصمیم‌گیری به پایتون.

## اهداف یادگیری کلیدی

- نوشتن دستورات شرطی ساده
- استفاده از if، elif و else برای انشعاب
- ترجمه شبه‌کد تصمیم به پایتون
- اشکال‌زدایی خطاهای تورفتگی و منطق
