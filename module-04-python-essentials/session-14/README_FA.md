# جلسه ۱۴: حلقه‌ها – while و for

## مرور جلسه

تکرار را با حلقه‌های while و for بررسی می‌کنیم، درک اینکه چه زمانی و چگونه از هر نوع حلقه استفاده کنیم.

## اهداف یادگیری کلیدی

- توضیح اینکه چرا حلقه‌ها مورد نیاز هستند
- استفاده از حلقه‌های while برای تکرار کنترل شده توسط شرط
- استفاده از حلقه‌های for با range() و دنباله‌ها
- اجتناب از اشتباهات رایج حلقه بی‌نهایت
