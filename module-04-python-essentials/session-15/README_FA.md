# جلسه ۱۵: کار با رشته‌ها

## مرور جلسه

عمیقاً به دستکاری رشته شیرجه می‌زنیم، یادگیری ایندکس‌گذاری، برش، روش‌ها و الگوهای پردازش متن.

## اهداف یادگیری کلیدی

- ایجاد و دستکاری رشته‌ها
- استفاده از روش‌های پایه رشته (len، lower، upper، slicing)
- تکرار روی کاراکترهای رشته
- حل وظایف ساده پردازش متن
