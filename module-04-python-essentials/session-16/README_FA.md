# جلسه ۱۶: مجموعه‌ها – لیست‌ها، تاپل‌ها، دیکشنری‌ها، مجموعه‌ها

## مرور جلسه

انواع مجموعه پایتون را برای ذخیره مقادیر متعدد بررسی می‌کنیم: لیست‌ها، تاپل‌ها، دیکشنری‌ها و مجموعه‌ها.

## اهداف یادگیری کلیدی

- توصیف اینکه چه زمانی از هر نوع مجموعه استفاده کنیم
- ایجاد و دسترسی به عناصر در لیست‌ها و دیکشنری‌ها
- تکرار روی مجموعه‌ها با حلقه‌های for
- استفاده از عملیات ساده مجموعه
