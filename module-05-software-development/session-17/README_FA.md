# جلسه ۱۷: توابع – بلوک‌های قابل استفاده مجدد کد

## مرور جلسه

یاد می‌گیریم توابع قابل استفاده مجدد بنویسیم، درک پارامترها، مقادیر بازگشتی و سازماندهی توابع.

## اهداف یادگیری کلیدی

- تعریف و فراخوانی توابع ساده
- استفاده از پارامترها و مقادیر بازگشتی
- درک حوزه محلی در برابر جهانی
- نوشتن توابع که الگوریتم‌ها را کپسوله می‌کنند
