# جلسه ۱۸: مدیریت خطای اساسی و اشکال‌زدایی

## مرور جلسه

یاد می‌گیریم خطاها را به طور优雅 با try/except مدیریت کنیم و استراتژی‌های اشکال‌زدایی برای یافتن و رفع مشکلات توسعه دهیم.

## اهداف یادگیری کلیدی

- تشخیص پیام‌های خطای رایج پایتون
- استفاده از try/except برای مدیریت خطای ساده
- اعمال استراتژی‌های اشکال‌زدایی اساسی
- پذیرش خطاها به عنوان فرصت‌های یادگیری
