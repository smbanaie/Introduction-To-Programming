# جلسه ۱۹: کار با فایل‌ها – خواندن و نوشتن فایل‌های متنی

## مرور جلسه

یاد می‌گیریم از فایل‌های متنی بخوانیم و در آنها بنویسیم، مهارتی crucial برای پردازش داده و پایداری.

## اهداف یادگیری کلیدی

- باز و بسته کردن فایل‌های متنی در پایتون
- خواندن محتویات فایل خط به خط
- نوشتن متن در فایل‌ها
- استفاده از الگوهای پایه پردازش فایل
