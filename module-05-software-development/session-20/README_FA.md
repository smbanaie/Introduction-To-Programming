# جلسه ۲۰: سازماندهی کد – ماژول‌ها و یک برنامه کوچک چند فایلی

## مرور جلسه

یاد می‌گیریم کد را در چندین فایل با استفاده از ماژول‌ها و importها تقسیم کنیم، برنامه‌های قابل نگهداری‌تری ایجاد کنیم.

## اهداف یادگیری کلیدی

- import توابع از فایل‌های پایتون دیگر
- سازماندهی کد در ماژول‌های منطقی
- ساختار پروژه‌های ساده چند فایلی
- درک استفاده از __main__
