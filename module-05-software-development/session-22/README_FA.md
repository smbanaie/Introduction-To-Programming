# جلسه ۲۲: مرور، تثبیت و پروژه‌های کوچک capstone

## مرور جلسه

جلسه نهایی ما کل دوره را مرور می‌کند، یادگیری را تثبیت می‌کند و فرصتی برای کار روی یک پروژه capstone شخصی به شما می‌دهد.

## اهداف یادگیری کلیدی

- مرور مفاهیم کلیدی دوره
- تکمیل یا ارائه یک پروژه کوچک
- تأمل در پیشرفت یادگیری
- شناسایی مراحل بعدی در برنامه‌نویسی
